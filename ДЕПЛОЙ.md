# 🚀 Деплой на Railway — финальная инструкция

## Файлы которые нужно загрузить

Собери в одну папку эти 4 файла:
- `bot.py`
- `requirements.txt`
- `railway.toml`
- `credentials.json` (скачанный с Google Cloud)

## Шаг 1 — Загрузи файлы на GitHub

1. Открой [github.com](https://github.com) → войди или создай аккаунт
2. Нажми **New repository** → назови `synergyhub-bot` → **Create**
3. Нажми **uploading an existing file** → перетащи все 4 файла → **Commit changes**

## Шаг 2 — Задеплой на Railway

1. Открой [railway.app](https://railway.app) → **Login with GitHub**
2. **New Project → Deploy from GitHub repo**
3. Выбери репозиторий `synergyhub-bot`
4. Railway начнёт сборку — подожди 1-2 минуты

## Шаг 3 — Добавь переменные окружения

В Railway открой проект → вкладка **Variables** → добавь три переменные:

| Name | Value |
|---|---|
| `BOT_TOKEN` | `8809258644:AAH9b_sCeoTofQkQyxVcuoNXr2sVNdQ376M` |
| `ADMIN_ID` | `127425552` |
| `SHEET_ID` | `1f6l1T3sWgesiBEJPUg4a_Dp60AezvrVGOL1848Q4tEo` |

После добавления Railway автоматически перезапустит бота.

## Шаг 4 — Проверь

Открой Telegram → найди своего бота → напиши `/start` → заполни тестовую заявку → проверь что она появилась в Google Sheets и пришло уведомление тебе в Telegram.

## ⚠️ Важно после настройки

Пересоздай токен бота — он был виден в чате:
1. Открой @BotFather → `/mybots` → выбери бота
2. **API Token → Revoke current token**
3. Скопируй новый токен
4. Обнови переменную `BOT_TOKEN` в Railway
